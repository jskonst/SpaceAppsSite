Ivanovo Python 2012


Приглашаем принять участие в работе конференции

Приглашаем всех желающих принять участие в работе конференции Ivanovo Python 2012. Конференция будет проходить 20-го октября 2012 г. в Ивановской государственной текстильной академии (ИГТА), начало в 10.00 по МСК. Для участия в работе конференции необходимо пройти регистрацию на сайте.

"Знакомтесь - Python"

Основной целью проведения конференции является ознакомление широкого круга специалистов - от программистов до научных работников с возможностями языка Python. Мы расскажем как используем этот язык программирования в научных вычислениях, разработке программ с графическим интерфейсом, в робототехнических устройствах для управления LegoNXT, связи с Arduino и т.д. Особенно широко будет рассмотрено применение языка Python в web-программировании.
